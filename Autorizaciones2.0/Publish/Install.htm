Instrucciones de instalación de la aplicación Microsoft Visual Studio LightSwitch

 Los siguientes pasos son necesarios para instalar una aplicación cliente de escritorio de dos niveles:

1.    Instale SQL Server Express en el equipo de destino (cliente) si aún no lo está. Las aplicaciones de LightSwitch requieren SQL Server 2005 o una versión posterior, o bien SQL Server Express 2005 o una versión posterior. Más información sobre opciones de instalación de SQL Server Express.

2.    En el equipo de destino, implemente el esquema de la base de datos mediante la herramienta de comando sqlcmd.exe. Esta herramienta se instala con SQL Server, normalmente en el directorio \Microsoft SQL Server\100\Tools\Binn en la carpeta Archivos de programa o Archivos de programa (x86).

a.     En un símbolo del sistema con privilegios elevados, cambie al directorio que contiene la salida de publicación y especifique un comando parecido al siguiente:

 sqlcmd.exe -i MyApplication.sql -S .\SQLExpress

 Reemplace MyApplication por el nombre de su aplicación y .\SQLExpress por la ubicación de la base de datos en el servidor.

3.    Busque el archivo web.config en el subdirectorio de archivos de aplicación de la salida publicada.

a.     Abra web.config en el Bloc de notas y edite la conexión de la sección <connectionStrings> de modo que se utilicen los nombres de su servidor y su base de datos. Por ejemplo, para la entrada parecida a:

<add name="_IntrinsicData" connectionString="Data Source=SERVERNAME\SQLEXPRESS;Initial Catalog=MyApplication;Integrated Security=True;Pooling=True;Connect Timeout=30;User Instance=False" />

Reemplace SERVERNAME por el nombre del servidor de bases de datos y MyApplication por el nombre de la aplicación.

4.    Ejecute Setup.exe en el equipo de destino para instalar la aplicación.
